# Lista de Exercícios

### 1. Par ou Ímpar (`par_impar`)
Verifica se um número inteiro é par ou ímpar.
* **Entrada:** Um número inteiro.
* **Saída:** Mensagem indicando "Par" ou "Ímpar".

### 2. Ordem Não Decrescente (2 números) (`ordena_dois`)
Lê dois valores e os exibe em ordem não decrescente.
* **Entrada:** Dois números inteiros.
* **Saída:** Os dois números ordenados.

### 3. Ordem Não Decrescente (3 números) (`ordena_tres`)
Lê três valores e os organiza de forma não decrescente.
* **Entrada:** Três números inteiros.
* **Saída:** A sequência dos três números ordenada.

### 4. Cálculo de Fatorial (`fatorial`)
Calcula o fatorial de um número N.
* **Entrada:** Um número inteiro **N**.
* **Observação:** Se **N <= 0**, o programa retorna **1**.
* **Saída:** O resultado final de $N!$.

### 5. Soma de N Valores (`soma_valores`)
Solicita uma quantidade específica de números e calcula a soma total.
* **Entrada:** 1. Um valor **n** (quantidade de números a serem lidos).
    2. Seguido de **n** números inteiros.
* **Observação:** Se **N <= 0**, o programa retorna **0**.
* **Saída:** O valor da soma de todos os números digitados.
